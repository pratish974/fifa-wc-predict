# fifa-wc-predict

A React + TypeScript app scaffolded with Create React App for FIFA World Cup predictions.

## Available scripts

- `npm install`
- `npm start`
- `npm test`
- `npm run build`

## Project structure

- `public/`: static HTML and assets
- `src/`: React components, pages, services, models, hooks, and styles
- `src/router/AppRouter.tsx`: application routing
- `src/firebase/firebase.ts`: Firebase configuration

## Notes

This project uses Create React App with TypeScript and includes a structure for predictions, leaderboards, and user management.
